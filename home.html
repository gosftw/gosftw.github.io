Projects

Multi-Object Recognition using CNNs

In this project we build and train a CNN model for recognizing multiple objects from different sizes. The model was trained on a dataset of various objects like cars, animals, and everyday objects, achieving high accuracy in identifying them.

I built the model using the Keras library in Python, with TensorFlow as the backend. The model consists of multiple convolutional layers, followed by max-pooling layers, classifying the objects from a Nx1x1 convolutional layer. With this architecture the model is capable to work with any size of image, and return an result for each visual field that was recognized by the model.

Real-time Speech-to-Text Transcription using CNNs

This project transcribing speech to text in real-time. The model is able to transcribe speech as it is being spoken, with a high level of accuracy, providing a smooth and seamless transcription experience.

The model consists of multiple convolutional layers, followed by a recurrent layers. The architecture was designed to extract features from the audio input and classify them into the corresponding text transcript.
